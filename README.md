## üöÄ Getting Started

### 1. Install

Add to your `pubspec.yaml`:

```yaml
dependencies:
  flutter_moon_ui_kit: ^0.1.0
````

Or from local path:

```yaml
dependencies:
  flutter_moon_ui_kit:
    path: ../flutter_moon_ui_kit
```

### 2. Import & Use

```
import 'package:flutter_moon_ui_kit/flutter_moon_ui_kit.dart';

MoonButton(
  label: "Click Me",
  onPressed: () {},
);

MoonTag(label: "Design");
```
